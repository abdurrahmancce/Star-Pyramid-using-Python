---

## 💻 Code

```python
n = 10

for i in range(1, n + 1):
    print(" " * (n - i) + "* " * i)
```

---

## ⚙️ How It Works

* `n` controls the height of the pyramid
* Loop runs from `1 → n`
* Leading spaces align the pyramid
* Stars increase line-by-line to form the shape
